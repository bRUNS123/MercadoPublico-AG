🔄 Sincronizar GEOPRO

Un “botón” para traer tus procesos a LicitaBoard sin copiar nada a mano. No instala nada — es solo un marcador.

1. Arrastra este botón a tu barra de marcadores:
🔄 Sincronizar GEOPRO
Si no ves la barra de marcadores: Ctrl+Shift+B para mostrarla.
2. Para usarlo:
  1. Inicia sesión en mercadopublico.cl y entra al Escritorio de Proveedor.
  2. Estando en esa pestaña, haz click en el marcador 🔄 Sincronizar GEOPRO.
  3. Se abre tu dashboard Mi MercadoPúblico ya sincronizado. Listo.
⚠️ Si te avisa que no encontró el token: a veces el navegador guarda el token de forma no accesible. En ese caso, abre DevTools (F12) → Network → una petición a servicios-escritorio → header authorization, copia el valor (sin “Bearer ”) y en el dashboard usa 🔑 Token → 📋 Pegar del portapapeles.

🔒 El token es tuyo y solo viaja entre la pestaña de MercadoPúblico y tu dashboard (se guarda en tu navegador, caduca ~8h). No se envía a terceros.
